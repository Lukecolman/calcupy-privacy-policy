Política de Privacidad de Calcupy

Última actualización: 14 de mayo de 2026

Esta Política de Privacidad describe cómo Calcupy recopila, utiliza y protege la información relacionada con el uso de la aplicación.

Calcupy es una aplicación diseñada para ayudar a los usuarios a calcular compras, convertir montos entre monedas, consultar cotizaciones, crear listas de productos y escanear montos o tickets mediante la cámara del dispositivo.

1. Responsable de la aplicación

La aplicación Calcupy es desarrollada y mantenida por Luke. Para consultas relacionadas con esta Política de Privacidad, podés contactarnos en: colman.html@gmail.com.

2. Información que no recopilamos

Calcupy no requiere la creación de cuentas de usuario y no solicita datos personales como nombre, documento de identidad, dirección, contraseña, información bancaria o datos de pago.

La aplicación no envía fotos, tickets, comprobantes o imágenes capturadas con la cámara a servidores propios. El análisis de montos o tickets se realiza únicamente con el objetivo de ayudar al usuario a detectar importes dentro de la aplicación.

3. Uso de la cámara

Calcupy puede solicitar acceso a la cámara del dispositivo para permitir el escaneo de montos, tickets o comprobantes. Este permiso se utiliza exclusivamente para que el usuario pueda analizar visualmente un monto y cargarlo de forma más rápida en la aplicación.

Las imágenes capturadas o analizadas no se envían a servidores propios de Calcupy ni se utilizan para identificar personalmente al usuario.

4. Uso de internet

Calcupy puede utilizar conexión a internet para obtener cotizaciones, tipos de cambio o información relacionada con monedas y valores de referencia.

Esta conexión se utiliza para mejorar la precisión de los cálculos y brindar información actualizada al usuario. Calcupy no utiliza esta conexión para crear perfiles personales de usuario ni para vender información personal.

5. Listas y datos guardados en el dispositivo

Calcupy puede permitir al usuario crear y guardar listas de productos, montos, cálculos o configuraciones dentro de la aplicación.

Esta información se guarda localmente en el dispositivo del usuario. Calcupy no envía estas listas a servidores propios ni las utiliza para identificar al usuario.

Si el usuario elimina la aplicación o borra sus datos desde la configuración del dispositivo, esta información local puede perderse.

6. Publicidad

Calcupy puede mostrar anuncios dentro de la aplicación. Para ello, la aplicación puede integrar servicios de publicidad de terceros, como Google AdMob u otras plataformas similares.

Estos servicios pueden recopilar o utilizar ciertos datos técnicos del dispositivo, identificadores publicitarios, información de uso o datos aproximados necesarios para mostrar, medir y mejorar los anuncios.

El tratamiento de datos realizado por estos proveedores externos está sujeto a sus propias políticas de privacidad. Recomendamos revisar las políticas de privacidad de los servicios publicitarios utilizados.

7. Servicios de terceros

Calcupy puede utilizar servicios de terceros para funciones como obtención de cotizaciones, análisis técnico, publicidad o mejora del funcionamiento de la aplicación.

Estos servicios pueden procesar información técnica necesaria para prestar sus funciones, de acuerdo con sus propias políticas de privacidad.

8. Seguridad

Calcupy aplica medidas razonables para proteger la información utilizada por la aplicación. Sin embargo, ningún método de almacenamiento o transmisión electrónica es completamente seguro.

La aplicación está diseñada para minimizar la recopilación de datos personales y limitar el uso de información al funcionamiento necesario de sus características.

9. Privacidad de menores

Calcupy no está dirigida específicamente a niños menores de 13 años y no recopila intencionalmente información personal de menores.

Si un padre, madre o tutor considera que un menor proporcionó información personal mediante la aplicación, puede contactarnos para solicitar la revisión o eliminación de dicha información, en caso de corresponder.

10. Cambios en esta Política de Privacidad

Podemos actualizar esta Política de Privacidad ocasionalmente para reflejar cambios en la aplicación, requisitos legales o mejoras en nuestras prácticas.

Cualquier cambio será publicado en esta misma página, indicando la fecha de última actualización.

11. Contacto

Si tenés preguntas, comentarios o solicitudes relacionadas con esta Política de Privacidad, podés contactarnos en: colman.html@gmail.com.
